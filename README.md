# WillRogersParadoxExperimenter_App
WillRogersParadoxExperimenter: An app to experiment the Will Rogers Paradox.

## WillRogersParadoxExperimenter: Will Rogers Paradox Experimenting App: [Link](https://menashasenanayaka.shinyapps.io/willrogersparadoxexperimenter_app/)

The app is designed to experiment the Will Rogers Paradox in a user-friendly manner. The users can create two groups of data and move values in order to experiment the paradox.

Note:

* First, the values for Group A and Group B should be selected from the drop down menu. 
* See the mean values of the two groups.  
* Next, move one value from one group to another and see the change in the mean value. 
* A message will be displayed stating whether the Will Rogers Paradox was satisfied or not. 
* The users can change values and experiment several times.

Limitations:

* The app is designed only for two groups with a value range from 0 to 999. 
* The analysis is limited to calculating the mean.

This shiny app will be helpful to someone who has a curiosity about the Will Rogers Paradox. Anyone needing to experiment the Will Rogers Paradox using an interactive tool can take advantage of this shiny app. Even the users who are not familiar with obtaining statistical outputs from software can take the maximum benefit from this user friendly app.
